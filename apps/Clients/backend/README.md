# Clients Backend

Backend dedicado para la seccion de clientes.

## Endpoints para Postman

- `GET http://localhost:4100/health`
- `GET http://localhost:4100/api/v1/ping`
- `GET http://localhost:4100/api/v1/clients/status`
- `POST http://localhost:4100/auth/login`
- `POST http://localhost:4100/auth/register`
- `GET http://localhost:4100/auth/me`
- `POST http://localhost:4100/auth/logout`
