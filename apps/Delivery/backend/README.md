# Delivery Backend

Backend dedicado para la seccion de repartidores y entregas.

## Endpoints para Postman

- `GET http://localhost:4200/health`
- `GET http://localhost:4200/api/v1/ping`
- `GET http://localhost:4200/api/v1/delivery/status`
- `POST http://localhost:4200/auth/login`
- `POST http://localhost:4200/auth/register`
- `GET http://localhost:4200/auth/me`
- `POST http://localhost:4200/auth/logout`
